# slabpad
My own macropad for gaming and video editing, It'll have 16 keys(4by4) and one rotary encoder for volume.

# Specifications
 
<ul>
<li>16x Cherry MX Switches <br>
<li>1x XIAO RP2040 <br>
<li>6x Blank DSA Keycaps <br>
<li>4x M3x16 Bolt <br>
<li>4x M3 Heatset <br>
<li>1x EC11 Rotary Encoder</li>  
</ul>

Others:
<ul>
<li>KMK Firmware
<li>hackpad (Assembly).f3z
</ul>
